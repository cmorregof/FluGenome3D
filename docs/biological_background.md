# Biological background

Influenza A is an enveloped, segmented RNA virus. HA and NA are major surface glycoproteins and define common subtype labels such as H1N1 and H3N2. This project focuses on HA/NA because they connect naturally to AntigenLM/AntigenSDE, sequence-context analysis, codon usage and structure-aware visualization.

This project does not infer antigenicity, immune escape, vaccine candidates, pathogenicity, transmissibility or viral fitness.
